# Pokemon Infinite Fusion PT-BR Fan Translation v1.0.0

## English

Initial public fan-made Brazilian Portuguese translation release.

This release includes:

- Brazilian Portuguese language file.
- Script-level UI translation support.
- Pokedex text patch mappings.
- Outfit, hat, and hairstyle description mappings.
- PowerShell installer with backup and restore.
- Bilingual documentation in English and Portuguese.
- Final validation report.

This package does not include Pokemon Infinite Fusion itself. Download the game only from the official Pokemon Infinite Fusion Discord maintained by the developers.

## Português

Primeira release pública da tradução fanmade em Português do Brasil.

Esta release inclui:

- Arquivo de idioma em Português do Brasil.
- Suporte de tradução para textos de interface em scripts.
- Mapeamentos de tradução da Pokédex.
- Mapeamentos de descrições de roupas, chapéus e cabelos.
- Instalador em PowerShell com backup e restauração.
- Documentação bilingue em inglês e português.
- Relatório final de validação.

Este pacote não inclui o Pokemon Infinite Fusion. Baixe o jogo somente pelo Discord oficial de Pokemon Infinite Fusion mantido pelos desenvolvedores.
