# Pokemon Infinite Fusion PT-BR Mod v1.1.4

Stability hotfix for the PT-BR translation package.

## English

### Fixed

- Restored Nature names to the original English/default names.
- Restored the Options/configuration screen text and identifiers to the game default to avoid config-screen crashes.
- Updated the package documentation so Nature names and stability-sensitive configuration screens are listed as preserved in English.

### Notes

- This hotfix does not include Ruby gameplay logic patches. It keeps the project focused on translation/data files.
- Pokedex species/fusion descriptions remain future work and are not included in the active installer payload.
- If you updated the base game with the official updater `.bat`, run the PT-BR installer again before opening your save.

## Portugues

### Corrigido

- Nomes das Natures restaurados para os nomes originais/default em inglês.
- Tela de Options/configuração restaurada para o padrão do jogo para evitar crashes nessa área.
- Documentação atualizada para deixar claro que Natures e telas de configuração sensíveis ficam preservadas em inglês.

### Notas

- Este hotfix não inclui patches de lógica Ruby de gameplay. O projeto continua focado em arquivos de tradução/dados.
- Descrições de espécies/fusões da Pokédex continuam como trabalho futuro e não entram no payload ativo do instalador.
- Se você atualizou o jogo base pelo updater oficial `.bat`, rode o instalador PT-BR novamente antes de abrir seu save.
