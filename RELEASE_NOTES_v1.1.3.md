# Pokemon Infinite Fusion PT-BR Fan Translation v1.1.3

## English

This hotfix improves the uninstall flow.

### Changed

- `Desinstalar tradução` now prefers the original pre-translation backup in `PTBR_BACKUPS`, instead of simply restoring the newest backup.
- This makes uninstalling the PT-BR mod return the game files to English more reliably.
- The installer now recommends running the official game update `.bat` after uninstalling, so the official updater can verify and refresh the original game files.
- Save safety remains enabled: graphical uninstall backs up saves and resets the saved language value back to English.

### Downloads

- `InfiniteFusionPTBRInstaller-v1.1.3.exe`: recommended single-file graphical installer.
- `PTBR-Translation-BAT-v1.1.3.zip`: separate BAT/PowerShell package.

## Português

Este hotfix melhora o fluxo de desinstalação.

### Alterado

- `Desinstalar tradução` agora prefere o backup original anterior à tradução dentro de `PTBR_BACKUPS`, em vez de simplesmente restaurar o backup mais recente.
- Isso faz a desinstalação do mod PT-BR voltar os arquivos do jogo para inglês com mais segurança.
- O instalador agora recomenda rodar o `.bat` oficial de atualização do jogo depois de desinstalar, para o updater oficial verificar e refrescar os arquivos originais.
- A segurança de save continua ativa: a desinstalação pelo `.exe` cria backup dos saves e volta o idioma salvo para inglês.

### Downloads

- `InfiniteFusionPTBRInstaller-v1.1.3.exe`: instalador gráfico recomendado em arquivo único.
- `PTBR-Translation-BAT-v1.1.3.zip`: pacote BAT/PowerShell separado.
